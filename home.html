Check & Pay Your Traffic Challan



How to Enter Vehicle Number

Use correct format: State Code + RTO Code + Number

No spaces or special characters

Example: DL01AB1234

How to Pay Challan

Check challan using vehicle number

Select pending challan and proceed to payment

Pay using UPI / Card and download receipt

Safety Measures & Precautions

Always wear helmet / seatbelt

Follow traffic signals strictly

Avoid overspeeding

Keep documents valid (DL, RC, Insurance)
